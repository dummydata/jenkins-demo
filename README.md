jenkins-demo
============

Jenkins demo using Vagrant

### Dependencies

  * [Vagrant](http://www.vagrantup.com)

## How to Use

Run bundler:

    $ bundle install

Run librarian-puppet:

    $ librarian-puppet install (--verbose)

Start the vagrant box:

    $ vagrant up
